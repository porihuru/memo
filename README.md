#「仕事の偏食」を減らすという自己管理

「嫌いではない」は才能です。好きになる必要はありませんが、“嫌いじゃない状態”でいられると、毎日のしんどさは確実に減ります。

仕事も食べ物と同じで、「好きじゃないけど人並みにできる」は慣れで作れます。逆に「嫌い」を放置すると、関わりが雑になりやすく、周りから任されにくくなる。これは意地悪ではなく、組織が回るための自然な動きです。結果として“できる場面”が減り、自分が消耗していきます。

不満や問題提起は大事ですが、そこで止まらず「どう改善するか」を形にする力が差になります。好き嫌いに左右されず一定の成果を出せる人ほど、信頼されます。

だからこそ、嫌いを「個性」で固定せず、減らす工夫をする。内面の自己研鑽として大切です。

#仕事の勉強・資格勉強は「作業興奮」で

仕事や資格に直結する勉強は、「やる気が出たらやる」だと始まらない。
そこで使えるのが 作業興奮です。
始めたら集中とやる気が後から付いてくる仕組みです。

ポイントは 最初の2分を固定すること。

・テキストを開く

・問題を1問だけ解く

・手順を1つだけ試す
これだけでOKです。
目標は「完了」じゃなく “開始”すること。

乗ってきたら続ける。
乗らなければやめる。
それでも「今日も始めた」という実績が残り、次の日が楽になります。


先日、○○○の会議に出席したのですが、そこで飛び交う「会議の横文字」がよく分からず困りました。
最近はニュースでも横文字が増えており、調べながら読むこともあります。
今回は自分なりにインターネットで調べて整理してみましたので、よければ一度目を通してみてください。


A（同意・依頼・急ぎ）
アグリー（agree）
意味：賛成・同意
例：「その案でアグリーです」「そこはアグリー取れてる？」
ポイント：“全員合意” か “自分は賛成” か曖昧になりがち

アサイン（assign）
意味：担当に割り当てる
例：「この件、Aさんにアサインで」
ポイント：担当＝責任者まで含むことが多い

ASAP（アサップ）
意味：できるだけ早く（最優先とは限らない）
例：「ASAPで対応お願いします」
ポイント：期限が無いので「今日中？明日？」は確認推奨

B（進め方・開発・働き方）
アジャイル
意味：小さく作って早く回す開発/進め方
例：「ウォーターフォールよりアジャイルで回す」
ポイント：計画なしで突っ込む とは別物（短サイクル管理）

OJT
意味：現場で仕事しながら教育
例：「OJTでキャッチアップして」
ポイント：教える側の時間＝リソースが要る

C（会議運営・資料）
アジェンダ（agenda）
意味：議題、順番
例：「アジェンダは3点です」
ポイント：アジェンダが無い会議は散りやすい

サマリー（summary）
意味：要約
例：「最後にサマリーください」
ポイント：結論・決定事項・宿題が入るのが理想

MTG（エムティージー）
意味：会議
例：「明日MTG入れます」
ポイント：雑談/共有/決定、目的を添えると強い

D（人・立場・関係者）
アテンド（attend）
意味：同席・参加・案内
例：「次回は部長アテンドで」
ポイント：「連れてくる」の意味で使われがち

ステークホルダー（stakeholder）
意味：利害関係者（影響を受ける人）
例：「ステークホルダーに確認が必要」
ポイント：決裁者/利用者/現場/監査など広い

アンバサダー（ambassador）
意味：推進役・広報役（顔になる人）
例：「現場アンバサダーを立てたい」
ポイント：権限は無いが影響力を期待される

インフルエンサー
意味：影響力がある発信者
例：「インフルエンサー起用で拡散」
ポイント：案件色が強すぎると逆効果も

E（判断・承認・上申）
エスカレーション（escalation）
意味：上位者/上位組織に判断を上げる
例：「この件はエスカレーションします」
ポイント：丸投げ ではなく「論点・選択肢」を添える

オーソライズ（authorize）
意味：承認する、権限付与
例：「その変更、誰がオーソライズ？」
ポイント：決裁者確認の言い換え

ディシジョン（decision）
意味：決定
例：「今日のディシジョンはこれ」
ポイント：決めたこと と 宿題 を分ける

F（根拠・証拠・事実）
エビデンス（evidence）
意味：根拠資料、証跡
例：「エビデンスありますか？」
ポイント：口頭ではなく“残るもの”が求められる

G（外部・市場・顧客）
インバウンド
意味：外から来る（問い合わせ/訪日/流入）
例：「インバウンドの問い合わせ増えた」
ポイント：対義語はアウトバウンド（こちらから打つ）

セグメント（segment）
意味：市場/顧客層の切り分け
例：「セグメント別に施策を変える」
ポイント：誰に効かせる話かが明確になる

ニッチ（niche）
意味：すき間市場
例：「ニッチだけど刺さる層がいる」
ポイント：小さいが強いことがある

H（契約・製造・体制）
OEM
意味：他社ブランド製造（相手先ブランド生産）
例：「OEM供給で展開」
ポイント：自社で作らず他社名で出るケース

アライアンス（alliance）
意味：提携
例：「アライアンス組めないか」
ポイント：契約・責任分界が肝

スキーム（scheme）
意味：仕組み、枠組み（やり方の設計）
例：「運用スキームを固める」
ポイント：怪しい 文脈で使う人もいるが、会議では「枠組み」意味が多い

I（課題・論点・施策）

ドクトリン（doctrine）
意味：基本思想・行動原則（判断の“根っこ”）
例：「うちのドクトリンは現場優先」「ドクトリンに沿うと今回はA」
ポイント：ポリシー（具体ルール）より上位の概念として使われがち。**例外が増えた時の“よりどころ”**にされる。

イシュー（issue）
意味：論点/課題（今、潰すべきコア）
例：「イシューは2つ。AとB」
ポイント：問題全部 ではなく“本丸”を指すことが多い

イニシアチブ（initiative）
意味：主導権、主導施策
例：「こちらがイニシアチブ取ります」
ポイント：責任もセットで背負うニュアンス

オポチュニティ（opportunity）
意味：好機/商機
例：「それはオポチュニティです」
ポイント：やる理由 を作る言葉

J（スケジュール・期限）
オンスケ（on schedule）
意味：予定通り
例：「オンスケで進んでます」
ポイント：根拠（進捗率、完了物）があると信用される

デッドライン（deadline）
意味：締切
例：「デッドラインは金曜」
ポイント：提出 か 完了 か確認

リスケ（reschedule）
意味：日程変更
例：「会議リスケお願いします」
ポイント：代替案（いつなら可）を添えると早い

マイルストーン（milestone）
意味：中間目標
例：「次のマイルストーンは要件確定」
ポイント：期限と成果物がセット

バッファ（buffer）
意味：余裕、予備
例：「バッファ1日見ましょう」
ポイント：無い計画は高確率で燃える

K（数値管理・成果指標）
KPI
意味：重要業績評価指標（途中の指標）
例：「KPIは問い合わせ件数」
ポイント：活動の結果 を追う

KGI
意味：最終目標指標
例：「KGIは売上/達成率」
ポイント：KPI→KGIにつながってるかが問われる

コンバージョン（conversion）
意味：成果（購入/申込/登録など）
例：「CVが伸びない」
ポイント：何をCVと定義してるかで話が変わる

L（品質・改善・仕上げ）
フィードバック
意味：意見・戻し
例：「フィードバックください」
ポイント：“何を見て”の条件を付けると良い

ブラッシュアップ
意味：磨き込む
例：「資料ブラッシュアップします」
ポイント：修正点（見せ方/数字/構成）を明確に

フィックス（fix）
意味：確定する
例：「この内容でフィックス」
ポイント：以降の変更ルールも決めると強い

M（コスト・予算・価値）

インセンティブ（incentive）
意味：動機づけの仕組み（報酬・評価・得になる条件）
例：「インセンティブ設計を見直そう」「インセンティブがズレてると動かない」
ポイント：お金だけじゃなく、評価・表彰・権限・手間削減も含む。KPIと噛み合わないと逆効果。

バジェット（budget）
意味：予算
例：「バジェット確保できる？」
ポイント：年度/枠/使途の確認

コスパ
意味：費用対効果
例：「コスパ悪いからやめよう」
ポイント：費用＝金/時間/人手も含む場合あり

コストリダクション
意味：コスト削減
例：「コストリダクション案ある？」
ポイント：品質/リスクとのトレードオフ

ペイ（pay）
意味：元が取れる（回収できる）
例：「それペイするの？」
ポイント：期間（いつ回収？）がセット

ベネフィット（benefit）
意味：得られる利益・効果
例：「ベネフィットは何？」
ポイント：誰のベネフィットかを明確に

N（競争・市場変化）
コンペティター（competitor）
意味：競合
例：「コンペティターはどこ？」
ポイント：比較軸（価格/品質/納期）

コモディティ化
意味：差別化が消え、どれも同じ扱いになる
例：「もうコモディティ化してる」
ポイント：価格勝負になりやすい

キャズム（chasm）
意味：新しいものが普及する際の“溝”
例：「キャズム越えが課題」
ポイント：最初の熱狂→一般普及の壁

O（体制・能力・人の量）
キャパ（capacity）
意味：処理能力、人員余力
例：「キャパが足りない」
ポイント：人数だけでなく“熟練度”も含む

リソース（resource）
意味：人/時間/予算/設備
例：「リソース追加必要」
ポイント：何が足りないかを特定（人？時間？）

人月（にんげつ）

意味：工数（1人×1か月）
例：「この改修、2人月です」
ポイント：2人で1か月＝1人で2か月 にならないことも（並列化限界）

ナレッジ / ノウハウ
意味：知識 / 実務のコツ
例：「ナレッジ共有」「ノウハウ残して」
ポイント：ナレッジ＝情報、ノウハウ＝手順・勘所

P（状態・保留・押し引き）
ペンディング（pending）
意味：保留
例：「その件はペンディング」
ポイント：再開条件（何が揃えば？）を決めないと永久保留

プッシュ（push）
意味：強めに進める／促す
例：「先方にプッシュします」
ポイント：関係性を壊さない範囲で

タイト（tight）
意味：余裕がない
例：「スケジュールタイトです」
ポイント：何が原因（人/期限/仕様増）かセットで

タスク（task）
意味：やること
例：「タスク切って管理しよう」
ポイント：担当・期限・完了条件まで

デッドライン（再掲）
意味：締切
例：「デッドライン動かせない」
ポイント：最優先事項の宣言

Q（共有・外注・運用）
シェア（share）
意味：共有
例：「資料シェアします」
ポイント：場所（Teams/メール）と期限を添える

クラウドソーシング
意味：外部に業務委託を募る
例：「クラウドソーシングで試作」
ポイント：品質管理と守秘が課題

ドライブ（drive）
意味：推進する
例：「この施策をドライブします」

R（広報・マーケ寄り）

バズマーケティング / バズる

ユーザビリティ

オルタナティブ（alternative）
