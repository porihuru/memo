「自分は優秀だ」と思っている人ほど、気をつけたいこと」

廊下を歩いていると、いろいろな人がいます。
足元を見ながら、何か考え込み、悩んでいるように歩く人。
一方で、前を向いて、笑顔で颯爽と歩いている人もいます。

ぱっと見れば、笑顔で軽やかに歩いている人のほうが余裕があり、うまくいっているように見えるかもしれません。
逆に、下を見ながら歩いている人は、何かに苦しみ、立ち止まりそうに見えることもあります。

ですが、いつもそうとは限りません。

登山を思い浮かべるとわかりやすいです。
山を登るとき、人は足元をよく見ます。
転ばないように、滑らないように、一歩一歩を確かめながら進みます。
登りは苦しく、息も上がります。
決して軽やかではありません。
けれど、その苦しい一歩一歩は、確実に頂上へ向かっています。

反対に、山を下るときはどうでしょうか。
登りよりも足取りが軽く、颯爽と進めることがあります。
風を受けて、余裕があるようにも見えます。
ただ、それは頂上に向かっている姿ではありません。
むしろ、頂上から遠ざかっている姿です。

人も同じなのだと思います。

悩みながら、苦しみながら、下を見て歩いている時期があります。
自分の力不足を感じたり、思うようにいかず、考え込みながら進む時期です。
見た目には、余裕がなく、格好よくも見えないかもしれません。
しかし、その時期は案外「登り」なのです。
苦しいけれど、確実に上へ向かっている時間です。

逆に、自分は優秀だ、自分はわかっている、自分はもう十分できていると思い始めると、人は足元を見なくなります。
確認をしなくなる。
人の助言を聞かなくなる。
自分を疑わなくなる。
すると、見た目は颯爽としていても、実は下り坂に入っていることがあります。

本当に優秀な人は、そのことを知っています。
だからこそ、調子のいい時ほど慎重です。
うまくいっている時ほど、足元を確かめます。
自分はまだ足りないところがある、と考えられる。
人の意見を素直に聞ける。
確認を怠らない。
その謙虚さがあるから、また次の頂上を目指せます。

一方で、自分を優秀だと思い込みすぎる人は、苦しんでいる人を見て「余裕がない」「能力が低い」と見がちです。
しかし、苦しんでいる人が上り坂にいて、余裕のあるように見える人が下り坂にいることもある。
そこを見誤ってはいけません。

仕事でも人生でも、いつも楽に進める時が成長している時とは限りません。
むしろ、悩み、迷い、苦しみながら進んでいる時のほうが、後から振り返れば一番伸びていた、ということはよくあります。

足元を見ながら歩いている自分を、情けないと思わなくていい。
それは登っている姿かもしれません。
そして、颯爽と歩けている時ほど、「自分はいま登っているのか、下っているのか」と一度考えたほうがいい。

自分は優秀だと思った時こそ、気をつけたい。
楽に見える時こそ、足元を確かめたい。
人の成長は、見た目の余裕では測れません。

苦しみながら下を向いて歩いている時は、案外、頂上に向かっている。
軽やかに進めている時は、知らないうちに頂上から遠ざかっていることもある。
