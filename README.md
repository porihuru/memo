「自分は優秀だ」と思っている人ほど、気をつけたいこと」

廊下を歩いていると、いろいろな人がいます。
足元を見ながら、何か考え込み、悩んでいるように歩く人。
一方で、前を向いて、笑顔で颯爽と歩いている人もいます。

ぱっと見れば、笑顔で軽やかに歩いている人のほうが余裕があり、うまくいっているように見えるかもしれません。
逆に、下を見ながら歩いている人は、何かに苦しみ、立ち止まりそうに見えることもあります。

ですが、いつもそうとは限りません。

登山を思い浮かべるとわかりやすいです。
山を登るとき、人は足元をよく見ます。
転ばないように、滑らないように、一歩一歩を確かめながら進みます。
登りは苦しく、息も上がります。
決して軽やかではありません。
けれど、その苦しい一歩一歩は、確実に頂上へ向かっています。

反対に、山を下るときはどうでしょうか。
登りよりも足取りが軽く、颯爽と進めることがあります。
風を受けて、余裕があるようにも見えます。
ただ、それは頂上に向かっている姿ではありません。
むしろ、頂上から遠ざかっている姿です。

人も同じなのだと思います。

悩みながら、苦しみながら、下を見て歩いている時期があります。
自分の力不足を感じたり、思うようにいかず、考え込みながら進む時期です。
見た目には、余裕がなく、格好よくも見えないかもしれません。
しかし、その時期は案外「登り」なのです。
苦しいけれど、確実に上へ向かっている時間です。

逆に、自分は優秀だ、自分はわかっている、自分はもう十分できていると思い始めると、人は足元を見なくなります。
確認をしなくなる。
人の助言を聞かなくなる。
自分を疑わなくなる。
すると、見た目は颯爽としていても、実は下り坂に入っていることがあります。

本当に優秀な人は、そのことを知っています。
だからこそ、調子のいい時ほど慎重です。
うまくいっている時ほど、足元を確かめます。
自分はまだ足りないところがある、と考えられる。
人の意見を素直に聞ける。
確認を怠らない。
その謙虚さがあるから、また次の頂上を目指せます。

一方で、自分を優秀だと思い込みすぎる人は、苦しんでいる人を見て「余裕がない」「能力が低い」と見がちです。
しかし、苦しんでいる人が上り坂にいて、余裕のあるように見える人が下り坂にいることもある。
そこを見誤ってはいけません。

仕事でも人生でも、いつも楽に進める時が成長している時とは限りません。
むしろ、悩み、迷い、苦しみながら進んでいる時のほうが、後から振り返れば一番伸びていた、ということはよくあります。

足元を見ながら歩いている自分を、情けないと思わなくていい。
それは登っている姿かもしれません。
そして、颯爽と歩けている時ほど、「自分はいま登っているのか、下っているのか」と一度考えたほうがいい。

自分は優秀だと思った時こそ、気をつけたい。
楽に見える時こそ、足元を確かめたい。
人の成長は、見た目の余裕では測れません。

苦しみながら下を向いて歩いている時は、案外、頂上に向かっている。
軽やかに進めている時は、知らないうちに頂上から遠ざかっていることもある。



敬語は「難しい」のではなく「役割」が違う

「よく、おられますか？ 基本的に間違ってます」

こう言うと少しきつく聞こえるかもしれませんが、敬語を覚えるうえでは大事な視点です。
実際、日常会話や職場で「部長は今、席におられますか」「先生は会場におられます」という言い方を耳にすることがあります。丁寧に話しているつもりでも、敬語の種類を混ぜてしまっているため、基本的には不自然な表現になります。

なぜかというと、「おる」は謙譲語だからです。
謙譲語は、自分や身内の動作をへりくだって表現し、相手を立てるための言葉です。たとえば「私は会場におります」「のちほど私が伺います」のように使います。
一方で、相手や目上の人の動作を高めて表現するのが尊敬語です。相手がいることを言いたいなら、「先生は会場にいらっしゃいます」が正しい形です。

つまり、相手を高めるべき場面で、自分を低くする言葉を使ってしまっているのが「おられますか」の違和感です。
「おる」に尊敬の意味を持つ「られる」を重ねた形ですが、土台が謙譲語なので、標準的な敬語としては適切ではありません。

敬語で大切なのは、言葉を難しくすることではありません。
誰の動作なのか、そしてその人を高めたいのか、自分をへりくだりたいのかを整理することです。これが分かると、敬語は一気に使いやすくなります。

⸻

尊敬語と謙譲語は「方向」が違う

敬語は大きく分けると、尊敬語と謙譲語があります。

尊敬語は、相手や目上の人の動作を高める言葉です。
たとえば、
	•	行く・来る・いる → いらっしゃる
	•	食べる・飲む → 召し上がる
	•	する → なさる
	•	見る → ご覧になる
	•	言う → おっしゃる
	•	会う → お会いになる
	•	聞く → お聞きになる

といった形になります。

一方、謙譲語は、自分の動作をへりくだって表現する言葉です。
たとえば、
	•	行く・来る → 参る、伺う
	•	いる → おる
	•	食べる・飲む・もらう → いただく
	•	する → いたす
	•	見る → 拝見する
	•	思う → 存ずる
	•	言う → 申す、申し上げる
	•	会う → お目にかかる
	•	聞く → 承る、伺う
	•	知らせる → お知らせする

となります。

ここで重要なのは、同じ動詞でも、相手に使うか自分に使うかで言い方が変わることです。

「課長はもう会社にいらっしゃいます」は尊敬語。
「私は先に会社に参っております」は謙譲語。

同じ「行く・来る・いる」に関する内容でも、主語が誰かで敬語は変わります。

⸻

よくある間違いは「丁寧にしたい気持ち」が原因

敬語の間違いは、失礼をしようとして起こるのではありません。
むしろ逆で、丁寧にしようとしすぎることで起こることが多いものです。

たとえば、
	•	× 部長は今、席におられます
	•	○ 部長は今、席にいらっしゃいます
	•	× 社長が先ほどそう申されました
	•	○ 社長が先ほどそうおっしゃいました
	•	× 先生の資料を拝見されましたか
	•	○ 先生の資料をご覧になりましたか

このように、謙譲語を相手に使ってしまうケースは非常に多く見られます。

「申す」「拝見する」「おる」は、自分側が使う言葉です。
相手に使うと、丁寧に見えても敬語の方向がずれてしまいます。

⸻

「二重敬語」より先に、「誰を立てるか」を考える

敬語の話になると「二重敬語はダメ」とよく言われます。
もちろんそれも大事ですが、実際の会話でまず意識したいのは、誰の動作を表しているかです。

敬語を使う前に、頭の中でこう整理すると間違いが減ります。
	•	相手・上司・お客様の動作なら → 尊敬語
	•	自分・自社・自分側の動作なら → 謙譲語
	•	文全体を丁寧にしたいなら → 丁寧語（です・ます）

この順番で考えるだけで、かなり自然になります。

たとえば電話でも、

「田中はただいま外出しております。戻りましたらご連絡いたします」

は正しい表現です。
田中が自分の会社の人間であれば、相手に対しては身内なので、へりくだって表現します。

反対に、

「鈴木部長は会議室にいらっしゃいます」

は、部長を立てる尊敬語です。

⸻

よく使う言葉ほど、正しく覚える価値がある

敬語は、試験のためだけの知識ではありません。
毎日の会話、電話、メール、報告、接客、会議など、仕事のあらゆる場面で使います。
だからこそ、よく使う言葉ほど正しく覚える価値があります。

特に覚えておくと役立つのは、次のような基本です。
	•	いる
相手 → いらっしゃる
自分 → おる
	•	行く・来る
相手 → いらっしゃる、おいでになる
自分 → 参る、伺う
	•	言う
相手 → おっしゃる
自分 → 申す、申し上げる
	•	見る
相手 → ご覧になる
自分 → 拝見する
	•	食べる・飲む
相手 → 召し上がる
自分 → いただく

この基本が身につくと、無理に難しい敬語をひねり出さなくても、十分に品のある話し方になります。

⸻

敬語は「知識」より「配慮」を伝える道具

敬語を使う本当の目的は、賢く見せることではありません。
相手への配慮を言葉にすることです。

そのためには、難しい表現をたくさん覚えるより、
「この動作は誰のものか」
「この場面では誰を立てるべきか」
を落ち着いて考えるほうが大切です。

「おられますか」のように、よく聞くけれど基本的には誤りとされる表現もあります。
ただ、それを笑うのではなく、
「丁寧にしたい気持ちは分かる。けれど正しくはこう言う」
と理解できれば、言葉の使い方は確実に上達します。

敬語は堅苦しいルールではなく、相手との距離を整えるための道具です。
正しく使えるようになると、会話は smoother になり、印象も信頼も変わります。
まずは一つ、“相手には尊敬語、自分には謙譲語”。
この原則だけでも、敬語はぐっと使いやすくなるはずです。
