「自分は優秀だ」と思っている人ほど、気をつけたいこと」

廊下を歩いていると、いろいろな人がいます。
足元を見ながら、何か考え込み、悩んでいるように歩く人。
一方で、前を向いて、笑顔で颯爽と歩いている人もいます。

ぱっと見れば、笑顔で軽やかに歩いている人のほうが余裕があり、うまくいっているように見えるかもしれません。
逆に、下を見ながら歩いている人は、何かに苦しみ、立ち止まりそうに見えることもあります。

ですが、いつもそうとは限りません。

登山を思い浮かべるとわかりやすいです。
山を登るとき、人は足元をよく見ます。
転ばないように、滑らないように、一歩一歩を確かめながら進みます。
登りは苦しく、息も上がります。
決して軽やかではありません。
けれど、その苦しい一歩一歩は、確実に頂上へ向かっています。

反対に、山を下るときはどうでしょうか。
登りよりも足取りが軽く、颯爽と進めることがあります。
風を受けて、余裕があるようにも見えます。
ただ、それは頂上に向かっている姿ではありません。
むしろ、頂上から遠ざかっている姿です。

人も同じなのだと思います。

悩みながら、苦しみながら、下を見て歩いている時期があります。
自分の力不足を感じたり、思うようにいかず、考え込みながら進む時期です。
見た目には、余裕がなく、格好よくも見えないかもしれません。
しかし、その時期は案外「登り」なのです。
苦しいけれど、確実に上へ向かっている時間です。

逆に、自分は優秀だ、自分はわかっている、自分はもう十分できていると思い始めると、人は足元を見なくなります。
確認をしなくなる。
人の助言を聞かなくなる。
自分を疑わなくなる。
すると、見た目は颯爽としていても、実は下り坂に入っていることがあります。

本当に優秀な人は、そのことを知っています。
だからこそ、調子のいい時ほど慎重です。
うまくいっている時ほど、足元を確かめます。
自分はまだ足りないところがある、と考えられる。
人の意見を素直に聞ける。
確認を怠らない。
その謙虚さがあるから、また次の頂上を目指せます。

一方で、自分を優秀だと思い込みすぎる人は、苦しんでいる人を見て「余裕がない」「能力が低い」と見がちです。
しかし、苦しんでいる人が上り坂にいて、余裕のあるように見える人が下り坂にいることもある。
そこを見誤ってはいけません。

仕事でも人生でも、いつも楽に進める時が成長している時とは限りません。
むしろ、悩み、迷い、苦しみながら進んでいる時のほうが、後から振り返れば一番伸びていた、ということはよくあります。

足元を見ながら歩いている自分を、情けないと思わなくていい。
それは登っている姿かもしれません。
そして、颯爽と歩けている時ほど、「自分はいま登っているのか、下っているのか」と一度考えたほうがいい。

自分は優秀だと思った時こそ、気をつけたい。
楽に見える時こそ、足元を確かめたい。
人の成長は、見た目の余裕では測れません。

苦しみながら下を向いて歩いている時は、案外、頂上に向かっている。
軽やかに進めている時は、知らないうちに頂上から遠ざかっていることもある。



AIとの付き合い方

　AIを本格的に使うようになって、気がつけば一年が経ちました。最初は「少し便利な道具」くらいに思っていましたが、今では日常のあちこちに入り込み、仕事でも生活でも、当たり前のように使う存在になっています。

　私が最初に助けられたのは、文書の添削でした。言い回しが固すぎないか、伝わりにくい表現になっていないか、誤字脱字はないか。以前なら何度も読み返して時間をかけていた作業が、AIを使うことで短時間で整理できるようになりました。自分では気づかない表現の癖も見つかり、文章を書くこと自体への苦手意識も少し薄れました。

　次に大きかったのは、プログラミングです。専門家でなくても、やりたいことを言葉で伝えると、たたき台を作ってくれる。もちろん、そのままで完璧に動くわけではありません。しかし「ゼロから考える」負担が減るだけで、作業の進み方は大きく変わります。わからないことを一つずつ調べていた頃に比べると、試すまでの速度が圧倒的に上がりました。これは仕事の効率化という意味でも、とても大きな変化でした。

　意外だったのは、自分の性格分析のような使い方です。考え方の傾向や、物事への向き合い方を言葉にして返してもらうと、自分を客観的に見るきっかけになります。人に相談するほどではないけれど、頭の中を整理したい。そんな時にもAIは役に立ちます。答えを決めてもらうのではなく、自分の考えを整理するための相手として使う。これはAIの意外な使い道かもしれません。

　しかも今は、特別な人だけが使う時代ではありません。ブラウザにもAIが付き、LINEにもXにもAIが付いています。意識して探しに行かなくても、日常の中にAIがすでに組み込まれています。使うか使わないかではなく、どう付き合うかを考える段階に入ったのだと思います。

　実際、AIによって時間が節約できているのは間違いありません。調べもの、文章の整理、発想の補助、作業の下準備。これまで自分一人で抱えていた時間の一部を、AIが肩代わりしてくれるようになりました。その分、本来考えるべきことや、最後の判断に時間を使えるようになります。忙しい日々の中で、この差はとても大きいものです。

　ただし、便利だからこそ気をつけたいこともあります。AIはもっともらしく間違うことがあります。自信ありげに答えていても、内容が正しいとは限りません。だからこそ、全部を任せるのではなく、最後は自分で確認することが大切です。AIは代わりに生きてくれる存在ではなく、あくまで補助してくれる道具です。地図は見せてくれても、進む道を決めるのは自分です。

　これから先、AIはさらに身近になっていくでしょう。若い世代だけのものでも、ITに詳しい人だけのものでもありません。20代でも50代でも、使い方次第で大きな力になります。大切なのは、怖がって遠ざけることでも、逆に過信することでもなく、便利さを活かしながら、最後は自分の頭で考えることです。

　AIとの付き合い方とは、結局のところ「任せる部分」と「自分で持つ部分」を見極めることなのだと思います。時間を節約し、視野を広げ、作業を助けてもらう。その一方で、判断と責任は手放さない。これからの時代に必要なのは、AIを使う力だけではなく、AIに使われない感覚なのかもしれません。
